# what is this repo for?

Get shadow for images 

# installation

`npm i shadowizard --save`

Then...

```
import {shadowizard} from 'shadowizard'

shadowizard({
    shadow_type: 'soft',
    padding: false,
})

```

# options

* *shadow_type* - _soft / hard (Default to soft)

* *padding* - _ boolean (Default to false)
